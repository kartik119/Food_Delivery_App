# 📖 Overview

Food Delivery App is a dynamic Java Full Stack web application that allows users to browse restaurants, explore menus, add food items to the cart, and place orders seamlessly.

The application is developed using Java, JSP, Servlets,  and MySQL following the MVC architecture to ensure scalability, maintainability, and clean code practices.

# 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/Ganesh-Basava-Nayak/Food_Delivery_App.git
```

---

### Import Project

Import the project into Eclipse IDE.

---

### Configure Apache Tomcat

- Install Apache Tomcat
- Add Tomcat Server in Eclipse
- Configure Runtime Environment

---

### Configure Database

Create a MySQL database.

Update database credentials in your configuration file.

---

### Run the Application

Start Tomcat Server and open:

```
http://localhost:8080/Food_Delivery_App
```
